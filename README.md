# Jan Pfeifer - Personal Website

This repository contains the source code for my minimalist personal website, hosted on GitHub Pages.

## Features

- Minimalist and elegant design
- Responsive layout for desktop and mobile
- Sections: About, Contact, Skills, Experience
- Easy to maintain and expand

## Deployment

1. Push the repository to GitHub.
2. Go to repository settings → GitHub Pages → set source to `main` branch `/root`.
3. The website will be live at `https://<your-username>.github.io/`.
